## Projects

**[Voxa](https://github.com/NischalGautam8/Voxa-voice-agent)** — AI voice agent that answers customer calls on your behalf, holds natural conversations, and saves structured summaries to your database. Built with Twilio, Deepgram, Cartesia, and Cerebras.

**[Second Brain](https://github.com/NischalGautam8/second-brain)** — Passively indexes your browsing activity into a local vector database, enabling semantic search over your own history.

**[Fit Tracker](https://github.com/NischalGautam8/fit-trackerpclient)** — Fitness activity logging application. Includes an **[Agentic CLI](https://github.com/NischalGautam8/fit-tracker-cli-agent)** that can autonomously manage activities and posts using LLMs.  
Related: **[Recommendation Engine](https://github.com/NischalGautam8/fittracker-xgboost-post-recommendation-engine)** — XGBoost-based system that recommends posts based on user activity patterns.

**[JobFit Agent](https://github.com/NischalGautam8/JobFit-Agent)** — Resume-job matching system using a fine-tuned BERT model to generate a confidence score (0–100) indicating candidate fit. Includes an agentic pipeline that iteratively improves resumes by editing LaTeX source to reach a target score.

**[Pixel Forge](https://github.com/NischalGautam8/pixel-forge)** — Image generation platform leveraging local models, ComfyUI workflows, and a job queue powered by Redis and BullMQ.
